# OpenAtmosphere

**Open-source hardware and software for sending real payloads into the atmosphere.**

OpenAtmosphere is a collaborative project focused on designing, building,
launching, and tracking real atmospheric payloads. The goal is to make
this kind of work accessible outside of institutional labs — open
designs, open firmware, open data.

## Why this project exists

Sending something real into the atmosphere has traditionally required
institutional resources: university programs, aerospace budgets,
specialized equipment. OpenAtmosphere aims to lower that barrier through
open, well-documented hardware and software that anyone can build from
and improve.

## Current status

This project is in its early stages. The repository currently contains
the initial structure and project vision. Core components are being
developed collaboratively — see the Issues tab for ways to contribute.

## Repository structure

- `firmware/` — Code that runs on the payload itself (sensors, GPS, data capture)
- `tracking/` — Ground-based tracking and recovery tools
- `ground-station/` — Telemetry reception and data logging
- `docs/` — Build guides, checklists, and reference documentation
